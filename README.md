# Albanisch Grammatiktrainer PWA

Erste Version einer offline-fähigen PWA für Albanisch A2.

## Enthalten
- 76 Grammatikthemen
- 900 Vokabeln aus `Vokabeln-neu-gesamt.xlsx`
- 3 feste Übungssitzungen
- Antworten und Fortschritt werden lokal im Browser gespeichert
- PWA-Manifest und Service Worker für Offline-Nutzung

## Lokal testen
Öffne `index.html` direkt im Browser. Für vollständige PWA-/Offline-Funktion am besten über einen kleinen lokalen Server starten, z. B.:

```bash
python3 -m http.server 8000
```

Dann im Browser öffnen:

```text
http://localhost:8000
```

## Auf dem Handy installieren
Die Dateien auf GitHub Pages, Netlify, Vercel oder einen Webspace hochladen. Dann den Link auf dem Handy öffnen und „Zum Startbildschirm hinzufügen“ wählen.
